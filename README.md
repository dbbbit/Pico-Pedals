# pico-pedals

### 简介

`pico-pedals` 项目将提供完整的赛车模拟 `踏板控制器` 方案，包括但不限于上位机软件、硬件资料及相关固件。

本项目支持免费商业授权及二次开发，只需提供品牌ID并保留原作者相关信息即可。

### 目录说明

- Doc：使用文档、接线图等
- Software：上位机 GUI 软件
- Firmware：配套固件

### 快速开始

这里先给最简单的使用芯片内案例，后续将提供更多外置ADC模块、称重传感器的使用文档。

接线图（以树莓派Pico为例，没错这就是项目名称的由来）

油门信号 -- GPIO_26
刹车信号 -- GPIO_27
离合信号 -- GPIO_28
VCC     -- ADC_VERF(3v3)
GND     -- AGND

- PICO自带 ADC 支持最大3v3输入，如果使用VSYS（5v）作为 VCC， 输入ADC前要用电阻分压到3V3以下。
- 踏板信号可以使用来自电位器、霍尔传感器或者是 INA122/AD62x 之类的放大器输出。

固件烧录

按住开发板上的BOOT按钮，插入USB数据线，保持片刻后释放按钮即可在文件管理器中看到一个可读写的存储器。
拖动 Firmware/rp2040 目录下的固件到存储器中即可完成烧录，之后就可以打开软件进行校准配置。

软件配置
校准：将每个踏板踩到底并保存最大值完成校准。
